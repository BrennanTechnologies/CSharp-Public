# CSharp-Public

## Summary

CSharp-Public is a collection of reusable C# libraries, utilities, and tools designed to accelerate .NET development. The repository includes:

- Extension methods for common .NET types
- Utilities for Azure, Graph, Identity, and more
- Visual Studio extension templates
- Build and packaging scripts
- Sample applications and demonstration projects

The project is organized into modular folders, each targeting a specific area such as Azure integration, core extensions, logging, state management, and more. It is intended for developers seeking to enhance productivity and code quality across a variety of C# and .NET projects.
